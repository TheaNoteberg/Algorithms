1 /**
2  * Segment Trees are an extremely useful data structure when dealing with ranges or intervals. They
3  * take O(n) time and space to construct, but they can do range updates or queries in O(log(n))
4  * time. This data structure is quite flexible; although the code below supports minimum and sum
5  * queries, these could be modified to perform other types of queries. This implementation uses lazy
6  * propagation (which allows for O(log(n)) range updates instead of O(n)). It should also be noted
7  * that this implementation could easily be modified to support coordinate compression (you should
8  * only have to change a few lines in the constructor).
9  *
10  * @author Micah Stairs
11  */
