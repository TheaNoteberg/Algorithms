1 /**
2  * An implementation of a hash-table using open addressing with quadratic probing as a collision
3  * resolution method.
4  *
5  * <p>In this implementation we are using the following probing function: H(k, x) = h(k) + f(x) mod
6  * 2^n
7  *
8  * <p>Where h(k) is the hash for the given key, f(x) = (x + x^2) / 2 and n is a natural number. We
9  * are using this probing function because it is guaranteed to find an empty cell (i.e it generates
10  * all the numbers in the range [0, 2^n) without repetition for the first 2^n numbers).
11  *
12  * @author William Fiset, william.alexandre.fiset@gmail.com
13  */
